# gulp-build-code-sharing
A build tool about gulp for you learn

## Run

```bash
# Step1: clone
git clone https://github.com/gongph/gulp-build-code-sharing.git

# Step2: install
npm i
```

## Build

- dev environment

```bash
npm run build:dev
```

- prod environment

```bash
npm run build:prod
```
